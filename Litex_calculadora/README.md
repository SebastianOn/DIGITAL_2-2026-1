
Esta carpeta corresponde al punto de entrada del proyecto. Desde aquí se ejecuta la aplicación principal y se integran los diferentes módulos que conforman la calculadora para que trabajen de forma conjunta.

# addsub_16 #

En esta carpeta se implementa el módulo encargado de realizar las operaciones de suma y resta de 16 bits. Además del diseño en Verilog, incluye los archivos de prueba (Testbench) y simulación para verificar que el circuito funcione correctamente antes de implementarlo en la FPGA.

# divisor #

Esta carpeta contiene el módulo encargado de la división. Aquí se encuentran los diferentes bloques que conforman el algoritmo de división, junto con el Testbench y los archivos de simulación utilizados para comprobar el funcionamiento del diseño.

Comentario general del proyecto

La estructura del proyecto está organizada por módulos, donde cada operación de la calculadora se desarrolla de manera independiente. Esto facilita el mantenimiento, la simulación y la integración de cada bloque dentro del sistema completo, además de hacer el código más ordenado y reutilizable.
